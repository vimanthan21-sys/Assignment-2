Project Description 

This project is a responsive, interactive website focused on Sustainable Development Goal 13 (Climate Action). The website aims to raise awareness about climate change, showcase its impacts, and encourage actionable steps for environmental sustainability. It includes three main sections: Home, Gallery, and About/Contact. 

Features 

Home Section: Hero banner with heading, brief description, and call-to-action button. 
Gallery Section: Image carousel displaying 5 climate-related images with captions. 
About/Contact Section: Information about SDG 13 and contact details with a toggle feature. 
JavaScript Carousel: Custom-built, automatic slideshow with next/previous navigation. 
jQuery Animations: Smooth scrolling and slideToggle effect for interactive elements. 
Responsive Design: Mobile-friendly layout using CSS Flexbox and media queries. 
Technologies Used 

HTML5 & Semantic Elements 
CSS3 (Flexbox, Grid, gradients, responsive design) 
JavaScript (Vanilla) for carousel functionality 
jQuery for animations and effects 
Purpose 

The website serves as an educational tool to increase awareness of climate change issues and demonstrate practical web development skills including interactive UI components and responsive design.
